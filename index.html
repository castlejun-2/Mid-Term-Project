개발목적
다양한 커스터마이징 제품이 인기를 끌고 있는 상황을 반영하여, 이러한 커스터마이징 주문 제작을 하는 서비스가 시장에 제대로 나와 있지 않음을 확인하고, 다양한 주문제작에 맞는 양식을 제공함으로써, 고객에게 편리함을 제공하기 위해 제작

주요기능
1. 커스터마이징 제품 등록
2. 커스터마이징 주문서 양식 제공
3. 지역별 필터링 및 검색 기능 제공
4. 판매자 실적 DashBoard 기능
5. 배너 이벤트 광고 기능
6. 인기 스토어 노출

개발목적
코로나 팬데믹으로 인해 주춤해진 다양한 온라인 만남을 위한 서비스 제공
ex) 스포츠(축구, 농구, e-sports), 취미(독서, 카공, 모각코 등)

주요기능
1. Filter 및 Search 기능을 통해 자신이 찾고자 하는 정보만을 검색
2. SSL Outh를 통한 Https로 통신
3. 최근 2주의 데이터만 노출시켜, 자신이 원하는 날짜에 올라와있는 만남 노출
4. 만나는 장소의 위치를 지도 표시
5. 자신이 작성한 만남들을 스케줄러 정리
6. 핸드폰 번호를 인증 구현
7. 프로필 사진 기능(default 포함)
8. Paging 기능 구현
9. 광고 배너 및 이벤트 구현

👊RESPECT👊

개발목적
개발자들은 포토폴리오를 위해서는, 일반적으로 깃허브에 프로젝트 코드를 올리고 노션에 포토폴리오를 정리한다.
이 때, 자신이 어떤 언어를 많이 사용하고 강점을 갖는지 알 수 없으므로 이러한 장점들을 분석하여 포토폴리오로 제작 할 수 있도록 제작
주요기능
1. 깃허브 레퍼지토리 분석
2. 다양한 개발자들의 포토폴리오 비교
3. 깃허브를 통한 소셜 로그인
4. 프로젝트 모집 및 참여
5. 서비스 자체 강의를 통한 수익창출
